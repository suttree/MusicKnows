Sie Template
=============

This is a simple site template that contains the basic ruby on rails code
for home, help, about, contact, register and login systems. It is intended
to make setting up a site a litle bit easier, since the boring stuff is
already done. This code has a frozen version of Rails and gems using

rake rails:freeze:gems
rake freeze_other_gems

Any new gems can be frozen by editing the file lib/tasks/gems.rake
